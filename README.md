Quizzler is an interactive quiz application built using **Python**, **Tkinter**, and **Object-Oriented Programming (OOP)**.  
The application fetches quiz questions dynamically from the **Open Trivia Database (OpenTDB) API** and presents them through a clean and user-friendly graphical interface.

The app provides real-time score tracking, visual feedback for correct and incorrect answers, and maintains a clear separation between quiz logic and UI components.

---

##  Features
- Graphical User Interface using Tkinter
- True / False quiz format
- Questions fetched from **Open Trivia Database (OpenTDB) API**
- Real-time score tracking
- Visual feedback for correct and wrong answers
- End-of-quiz summary
- Clean OOP-based architecture

---

##  API Used
- **Open Trivia Database (OpenTDB)**  
  A free and open trivia API providing questions across multiple categories and difficulty levels.
